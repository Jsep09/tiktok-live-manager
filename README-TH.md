# คู่มือการใช้งาน TikTok Live Viewer Manager
### สำหรับผู้ใช้ Windows — ฉบับภาษาไทย

---

## โปรแกรมนี้คืออะไร ทำอะไรได้บ้าง?

**TikTok Live Viewer Manager** คือโปรแกรมช่วย Host ที่ไลฟ์ TikTok ให้สามารถ **กดฟังเสียงอ่าน username** ของคนที่เข้ามาดูไลฟ์ได้ทันที

**ปัญหาที่แก้ได้:**
- อ่าน username ภาษาอังกฤษไม่คล่อง ไม่รู้จะออกเสียงยังไง
- อยากพูดถึงคนดูแต่จำชื่อไม่ได้ หรืออ่านไม่ออก

**สิ่งที่ทำได้:**
- เห็นรายชื่อคนที่เข้ามาดูไลฟ์แบบ real-time
- กดปุ่ม 🔊 ข้างชื่อ แล้วจะมีเสียงอ่าน username ออกมาเลย
- ค้นหาชื่อคนดูได้
- เห็นสถานะว่าใครยังดูอยู่ หรือออกไปแล้ว
- เห็นจำนวนคนดูทั้งหมด

---

## สิ่งที่ต้องติดตั้งก่อน

ต้องติดตั้งโปรแกรม 2 ตัวนี้ก่อน (ทำครั้งเดียวพอ):

### 1. Python
- ดาวน์โหลดที่: **https://www.python.org/downloads/**
- กดปุ่ม "Download Python" (เวอร์ชันล่าสุด)
- ตอนติดตั้ง **ต้องติ๊ก ✅ "Add Python to PATH"** ด้วย (สำคัญมาก!)

### 2. Node.js
- ดาวน์โหลดที่: **https://nodejs.org/**
- เลือกเวอร์ชัน **LTS** (ตัวซ้ายมือ แนะนำสำหรับผู้ใช้ทั่วไป)
- ติดตั้งตามปกติ กด Next ไปเรื่อยๆ ได้เลย
- **ต้องติ๊ก ✅ "Add to PATH"** ด้วย (ถ้าถามถึง)

> 💡 **หลังติดตั้งแล้ว ให้รีสตาร์ทคอมพิวเตอร์ 1 ครั้ง** เพื่อให้ระบบจำได้

## วิธีดาวน์โหลดโปรแกรม (ทำครั้งแรกครั้งเดียว)

### วิธีที่ 1: Download ZIP (ง่ายที่สุด ไม่ต้องใช้ Git)

1. ไปที่หน้า GitHub ของโปรแกรม
2. กดปุ่มสีเขียว **"Code"**
3. เลือก **"Download ZIP"**
4. แตกไฟล์ ZIP ไปยังโฟลเดอร์ที่ต้องการ เช่น `C:\Users\ชื่อคุณ\tiktok-live-manager`

### วิธีที่ 2: ใช้ Git Clone (ถ้ามี Git)

เปิด Command Prompt แล้วพิมพ์:
```
git clone https://github.com/[ชื่อ-repo].git
```

## วิธีเปิดโปรแกรม (ทุกครั้งที่ใช้)

1. เปิดโฟลเดอร์ที่เก็บโปรแกรม (`tiktok-live-manager`)
2. **ดับเบิ้ลคลิก** ที่ไฟล์ **`start.bat`**
3. รอสักครู่ — จะมีหน้าต่างสีดำเปิดขึ้น 2 หน้าต่าง (อย่าปิดนะ!)
4. **ครั้งแรก** ระบบจะติดตั้งส่วนเสริมต่างๆ อัตโนมัติ อาจใช้เวลา 2-5 นาที
5. หลังจากนั้น Chrome จะเปิดขึ้นมาอัตโนมัติที่หน้าโปรแกรม

> ⚠️ **หน้าต่างสีดำ 2 หน้าต่างนั้นคือตัวโปรแกรมที่กำลังทำงาน — ห้ามปิด!**
> ถ้าปิดแล้วโปรแกรมจะหยุดทำงาน

## วิธีใช้งานจริง

### ขั้นตอนที่ 1 — เปิดโปรแกรม
ดับเบิ้ลคลิก `start.bat` แล้วรอ Chrome เปิดขึ้นมา

### ขั้นตอนที่ 2 — เชื่อมต่อไลฟ์
1. ในหน้าโปรแกรม จะเห็นช่องกรอก username
2. พิมพ์ TikTok username ของตัวเอง เช่น `@myusername`
3. กดปุ่ม **Connect**
4. รอจนสถานะเปลี่ยนเป็น **"Connected"** (สีเขียว)

> 💡 ต้องกำลังไลฟ์อยู่จริงๆ ถึงจะเชื่อมต่อได้

### ขั้นตอนที่ 3 — ดูรายชื่อคนดู
- คนที่เข้ามาดูไลฟ์จะปรากฏในรายการอัตโนมัติ
- เห็นทั้ง username และชื่อที่แสดง
- แถบสีแสดงว่าใครยังดูอยู่ (สีสด) หรือออกไปแล้ว (สีจาง)

### ขั้นตอนที่ 4 — กดฟังเสียงอ่านชื่อ
- กดปุ่ม **🔊** ข้างชื่อคนที่ต้องการ
- จะได้ยินเสียงอ่าน username ออกมาทันที

### ขั้นตอนที่ 5 — ค้นหาชื่อ
- ถ้ามีคนดูเยอะ ใช้ช่อง **Search** พิมพ์บางส่วนของชื่อเพื่อค้นหา

### ขั้นตอนที่ 6 — หยุดการเชื่อมต่อ
- กดปุ่ม **Disconnect** ในโปรแกรมเมื่อเลิกไลฟ์

---

## วิธีปิดโปรแกรม

เมื่อต้องการปิดโปรแกรมทั้งหมด:

1. ปิด Chrome ตามปกติ
2. **ดับเบิ้ลคลิก** ที่ไฟล์ **`stop.bat`**
3. รอสักครู่ จะมีข้อความยืนยันว่าปิดเรียบร้อย

> ❌ **อย่าปิดแค่หน้าต่างสีดำ** เพราะอาจมี process ค้างอยู่เบื้องหลัง
> ให้ใช้ `stop.bat` เสมอ

## ปัญหาที่เจอบ่อย และวิธีแก้

### ❌ TTS ไม่มีเสียง กดปุ่ม 🔊 แล้วเงียบ

**สาเหตุและวิธีแก้:**
1. ตรวจสอบว่า volume ของคอมพิวเตอร์ไม่ได้ Mute
2. ตรวจสอบว่า volume ของ Chrome ไม่ได้ Mute (คลิกขวาที่ tab แล้วดู "Mute Site")
3. **ลองคลิกที่หน้าเว็บก่อน** แล้วค่อยกดปุ่ม 🔊 ใหม่ (browser ต้องการการ interact ก่อน)
4. **ใช้ Chrome** แทน browser อื่น
5. ถ้ายังไม่ได้ ลองปิด Chrome แล้วเปิดใหม่

---

### ❌ เชื่อมต่อไม่ได้ / สถานะไม่เปลี่ยนเป็น Connected

**สาเหตุและวิธีแก้:**
1. ตรวจสอบว่า **กำลังไลฟ์อยู่จริงๆ** บน TikTok
2. ลองกรอก username โดยไม่มีเครื่องหมาย `@` (เช่น `myusername` แทน `@myusername`)
3. ตรวจสอบว่าหน้าต่างสีดำ (Backend) ยังเปิดอยู่ ถ้าปิดไปแล้วให้รัน `start.bat` ใหม่
4. ลอง Disconnect แล้ว Connect ใหม่

---

### ❌ หน้าโปรแกรมแสดง Error หรือหน้าว่าง

**วิธีแก้:**
1. รอสักครู่ — บางครั้ง Backend ยังไม่ทันเริ่มทำงาน
2. กด F5 เพื่อ Refresh หน้า Chrome
3. ถ้ายังไม่ได้ ปิดทุกอย่างแล้วรัน `start.bat` ใหม่

---

### ❌ start.bat ขึ้นว่า "Python ไม่พบ" หรือ "Node ไม่พบ"

**วิธีแก้:**
1. ติดตั้ง Python หรือ Node.js ตามที่ระบุไว้ข้างต้น
2. **สำคัญ:** ต้องติ๊ก "Add to PATH" ตอนติดตั้ง
3. หลังติดตั้งแล้ว **รีสตาร์ทคอมพิวเตอร์** แล้วลองใหม่

---

### ❌ ครั้งแรกช้ามาก ใช้เวลานานเกิน

ปกติแล้วการติดตั้งครั้งแรกอาจใช้เวลา 3-10 นาที ขึ้นอยู่กับความเร็วอินเทอร์เน็ต
ครั้งต่อไปจะเร็วมาก (ไม่ต้องติดตั้งใหม่)

---

## คำแนะนำพิเศษ

### ✅ ใช้ Chrome เสมอ

โปรแกรมนี้ใช้ฟีเจอร์ **Text-to-Speech** ของ browser ในการอ่านชื่อ
Chrome รองรับได้ดีที่สุด — แนะนำให้ใช้ Chrome เสมอ

| Browser | TTS ทำงานได้ |
|---|---|
| ✅ Google Chrome | ดีที่สุด |
| ✅ Microsoft Edge | ดี |
| ⚠️ Firefox | เสียงน้อย อาจมีปัญหา |
| ❌ Safari | ไม่รองรับบน Windows |
