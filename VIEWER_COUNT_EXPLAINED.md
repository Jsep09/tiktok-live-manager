# ทำไมจำนวน Viewer ในแอปไม่ตรงกับ TikTok?

> เอกสารนี้อธิบายสาเหตุที่ตัวเลขใน TikTok Live Viewer Manager
> แตกต่างจากตัวเลขที่แสดงบน TikTok — และทำไมมันถึงเป็นเรื่องปกติ

---

## สรุปสั้นๆ

| ตัวเลข | ความหมาย |
|---|---|
| **ตัวเลขบน TikTok** | ผู้ชมทั้งหมด รวมคนที่นั่งดูเงียบๆ |
| **ตัวเลขในแอปนี้** | เฉพาะคนที่ทำ action บางอย่างใน Live |

ความต่างตรงนี้ **ไม่ใช่ bug** — เป็นข้อจำกัดจากฝั่ง TikTok โดยตรง

---

## สาเหตุที่ 1 — เห็นเฉพาะคนที่ทำ Action

ระบบจับได้เฉพาะ viewer ที่ทำสิ่งต่อไปนี้ระหว่าง Live:

- เข้ามาใหม่แล้ว **ส่ง join notification** ขึ้นมาบนจอ
- **comment** อะไรก็ตามในช่อง chat
- กด **like** (หัวใจ)
- ส่ง **ของขวัญ (gift)**

**คนที่นั่งดูเงียบๆ โดยไม่ทำอะไรเลย จะไม่ปรากฏในรายการ**

ตัวอย่างเช่น ถ้ามีคน 500 คนดูอยู่ แต่มีแค่ 80 คนที่ comment หรือ like
แอปนี้จะแสดงรายชื่อแค่ 80 คนนั้น ส่วนอีก 420 คนที่ดูเงียบๆ จะไม่เห็น

---

## สาเหตุที่ 2 — TikTok ส่งข้อมูลมาไม่ครบ

แม้จะเห็น join notification รัวๆ บน TikTok app แต่ข้อมูลที่ TikTok
ส่งให้ระบบภายนอก (ผ่าน unofficial API) จะถูก **ตัดทอนลงโดยอัตโนมัติ**

TikTok ไม่ได้ส่งทุก event ออกมา โดยเฉพาะช่วงที่มีคนเข้าพร้อมกันเยอะๆ
ระบบจะรับได้เพียงบางส่วนของ event เหล่านั้นเท่านั้น

---

## เปรียบเทียบให้ชัดขึ้น

```
TikTok แสดง: 500 viewers

แอปนี้เห็น: 80 คน
  ├─ comment            → 45 คน  ✅ เห็น
  ├─ like               → 20 คน  ✅ เห็น
  ├─ ส่ง join (ที่รับได้) → 15 คน  ✅ เห็น
  └─ ดูเงียบๆ            → 420 คน ❌ ไม่เห็น
```

---

## สิ่งที่ตัวเลขใน App บอกได้

แม้จะไม่ครบ แต่ตัวเลขในแอปนี้ยังมีประโยชน์:

- รู้ชื่อคนที่ **active จริงๆ** ใน Live
- กดอ่านชื่อ username คนที่ interact กับเราได้ทันที
- ดูว่า viewer คนไหนยังดูอยู่หรือออกไปแล้ว

**เป้าหมายของแอปนี้คือช่วยอ่านชื่อ — ไม่ใช่แทนที่ตัวเลข viewer บน TikTok**

---

## ข้อจำกัดนี้แก้ได้ไหม?

ไม่ได้ครับ — เป็นข้อจำกัดจาก TikTok โดยตรง

- TikTok **ไม่มี official API** สำหรับ viewer list
- Unofficial library (TikTokLive) ได้รับข้อมูลเท่าที่ TikTok ส่งออกมาเท่านั้น
- ไม่มีทางดึง "รายชื่อผู้ชมทั้งหมด" แบบ real-time ได้

> หากวันไหน TikTok เปิด official API ก็อาจแก้ได้ในอนาคต
